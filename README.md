# Weather Application

This project is a web-based weather application that fetches real-time weather data from the OpenWeatherMap API and displays weather information in a graphical format on a web page.

## Features

* Get real-time weather information
* Search weather by city name
* Display temperature, humidity, and weather conditions
* Weather data visualization using graphs
* Responsive user interface

## Technologies Used

* HTML5
* CSS3
* JavaScript (ES6)
* OpenWeatherMap API
* Chart.js

## How to Run

1. Download or clone the repository.
2. Open the project folder.
3. Open **index.html** in a web browser.
4. Enter a city name to view the weather information.
